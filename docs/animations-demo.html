Usage Examples

Basic Usage

<!-- Simple animation -->
<button className="animate-pulse">Click me</button>

<!-- With modifiers -->
<div className="animate-bounce animated-slow animated-infinite">
  Bouncing slowly forever
</div>

<!-- Hover animation -->
<button className="animate-pulse-hover">Hover me</button>

<!-- Icon animation -->
<svg className="icon-rotate">...</svg>

<!-- Button animation -->
<button className="button-pulse">Animated Button</button>

Available Modifiers

  • animated-fast - 0.5s duration
  • animated-slow - 2s duration
  • animated-slower - 3s duration
  • animated-infinite - Loop forever
  • animated-delay-1s - 1s delay before starting
  • animated-delay-2s - 2s delay before starting
  • animated-delay-3s - 3s delay before starting
